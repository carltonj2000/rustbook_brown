# Rust Language Notes

| Command                              | Details                          |
| ------------------------------------ | -------------------------------- |
| `rustup component add rust-analyzer` | add nvim editor tool             |
| `rustup doc`                         | open local copy of the rust docs |
| `cargo new dir`                      | create init project in dir       |
| `cargo init`                         | create a Cargo.toml init file    |
| `cargo build`                        | build                            |
| `cargo run`                          | run                              |
| `cargo check`                        | check to see if code compiles    |
